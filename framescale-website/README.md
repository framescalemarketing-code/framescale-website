# FrameScale Marketing Website

Multi page marketing website built with Next.js App Router and Tailwind CSS.

## Pages

- / Home
- /services
- /case-studies
- /pricing
- /about
- /contact
- /privacy
- /terms

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Notes

- Brand colors are defined in src/app/globals.css.
- Navigation items are defined in src/lib/site.ts.
- The contact form opens the visitor's email client with a prefilled message.
